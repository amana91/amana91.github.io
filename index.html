Bootcamp 2020: Flying Ping Pong

My first programming project, concluding the 2 week Programming Bootcamp at City (2020). This Processing project demonstrates a comprehensive understanding of the use of arrays, loops, and if statements. When you start the game you are taken to the main screen from there, you need to click on the screen and the game will begin.

You can then play the game and also see your score at the top of the screen which I aligned to be in the centre top. Everytime the ball hits the walls you will see the health bar at the top decrease and change colour from green to amber to red until the game ends. When the game ends you will be taken to the restart screen where you will be shown your final score and you can click the screen again if you want to restart the game.

Check out the code on Github: https://github.com/amana91/amana91.git/


Tom & Jerry Mini Game

This is my Java module game I created. It's a simple but fun game. There are 5 different levels each with different background images and sounds. One of the main challenges while coding this game was making multiple levels each with pickup points and ensuring that the positions of each body and object was correct which ensured the game would be successful. But through this challenge I was able to create a fun game and develop my Java skills further.

You can see the counter at the top of the game showing how many bottles are collected, you can also see as each levels advances it adds more challenges, pickups and new characters. There is also a GUI with the options to pause, resume and quit the game at the bottom.
